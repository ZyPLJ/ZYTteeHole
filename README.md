# ZY树洞
# 前言
ZY树洞是一个基于.NET Core开发的简单的评论系统，主要用于大家分享自己心中的感悟、经验、心得、想法等。
好了，不卖关子了，这个项目其实是上班无聊的时候写的，为什么要写这个项目呢？因为我单纯的想吐槽一下工作
中的不满而已。

# 项目介绍
项目很简单，主要功能就是提供一个简单的评论系统，用户可以发布自己的评论，然后评论以弹幕的形式展示在页面上。

# 项目特点
- 基于.NET Core开发，跨平台
- 简单易用，界面简洁
- 匿名评论，不用注册即可发表评论
- 弹幕效果，评论以弹幕的形式展示在页面上

# 主要技术
- ASP.NET Core Web API
- Entity Framework Core
- Sql Server数据库
- Vue.js
